bats
=======

`bats` is an R client to retrieve data from the [Bermuda Atlantic Time-Series Study (BATS)]() ftp server.

## Quick start

### Install

```r
# install.packages("devtools")
devtools::install_github("sckott/bats")
library("bats")
```
